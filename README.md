## Project: Set up

1. Clone repository
2. Open repository folder in Visual Studio Code and go to backend-api subfolder
2. Run command: `nmp i`
3. Rename ".env.example" to ".env"
4. Open .env file and insert correct values to the variables
4. Run the project using command `node .`
4. Open browser at given address.
